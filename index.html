One-time setup needed.
Takes about 5 minutes.

Step 1 — Create Google Sheet

Go to sheets.google.com → create a new blank sheet → name it Bodycraft Daily Report.

Copy the Sheet ID from the URL:
docs.google.com/spreadsheets/d/THIS_PART/edit

Step 2 — Set up Apps Script

In your sheet → Extensions → Apps Script → delete everything → paste the BodycraftAppsScript.js code → replace YOUR_GOOGLE_SHEET_ID_HERE with your Sheet ID → Save → select setupSheets → Run → approve permissions.

Step 3 — Deploy & paste URL here

In Apps Script → Deploy → New deployment → Web app → Execute as: Me → Access: Anyone → Deploy → copy the URL → paste below.
